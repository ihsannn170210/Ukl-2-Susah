# Program untuk mencari matriks
Program java untuk Menjumlahkan dua buah matriks berukuran sama, yang diinputkan oleh pengguna.

- Program meminta pengguna memasukkan jumlah baris dan kolom matriks.
- Pengguna mengisi elemen-elemen dari Matriks A.
- Pengguna mengisi elemen-elemen dari Matriks B.
- Program menjumlahkan setiap elemen dari Matriks A dan B, lalu menyimpannya di Matriks Hasil.
- Program menampilkan Matriks Hasil ke layar.
